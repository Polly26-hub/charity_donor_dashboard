# charity_donor_engagement_dashboard
## Project Overview
The objective of this project was to analyse was to analyse donor data and create and interactive dashboard that could support fundraising and CRM decision.making
## Project Objectives
- Analyse donor behaviour and find patterns.
- Monitoring fundraising performance using key performance indicators (KPI).
- Compare campaign performance.
- Identify the locations contributing the highest donation income.
- Visualise trends to support data-driven fundraising decisions.
- ## Dataset
The dataset contains 1,000 fictional donor records, created to simulate data that might be stored in a charity CRM system.

The dataset includes the following fields:
- Donor ID
- Donation Date
- Donation Amount
- Donation Type (Regular or One-off)
- Campaign
- Location
- Age Group

## Dashboard features

## Key Performance Indicators (KPIs)

- Total number of donors
- Total donation value
- Top performance location by donation amount
- Top performing campaign
  
The dashboard is interactive and allows users to filter data by location, with KPIs updating automatically.

## Visualisations
The dashboard includes:

- line chart showing donation over time
- bar chart comparing fundraising campaign performance and average amount per campaign
- bar chart displaying total donations per location
- pie chart comparing regular donors with one-off donors

## Key insights

  - The christmas campaign generated the highest total donation value.
  - One- off donations accounted for 80% of total income, highlighting a significant opportunity to increase long-term donor value.
  - London generated the highest total donation value across all locations, suggesting strong donor engagement in this region and potential for more focused fundraising activity.
  - Donations remained relatively consistent for all locations from January to November. There was a significant increase in December which suggests strong seasonal giving behaviour.
